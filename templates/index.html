It's All About Me

Hi I'm Liam.
I am a Full Stack Developer based in the UK with a background in the design.

Over the last year I have learnt how to bring my imagination to the web. I love creating eye catching application with UIs users can easily navigate.

I got into coding because I hated the limitions of Shopify themes and how little you could change. I am very much of the idea that if you can imagine something, it can be created! So I started taking a look under the hood and tinkering with what I could find. I learnt a lot about HTML and CSS but I really got into the nit and git when I enrolled in the Code Institute's Full Stack Software Development Diploma where I created E-commerce, Games, Web Applications and Sites.

One of my many reasons for getting into coding was for the problem solving. I love a challenge and coding provides an endless varity.

In my free time I enjoy snowboarding and I'm quite the avid baker.

Geometric

Geometric is an endless scroller where the movement the of the world and the objects around them give the simulation of movement. The game contains three types of obstacles. The three obstacles then have unique mechanics which the player has to interact with differently to pass. The back end of the game has an easy to use level mapper that allows developers to create and edit levels.

  • Custom Key Mapping
  • Character Customisation
  • Obstacle Variation each with Unique Mechanics
  • Product Filtering and Search
  • Varied Obstacle Courses with Level Mapping
  • Custom Error Pages
